# Head & Tails Game

## Description
An interactive coin-flip game where the player selects either head or tail and competes against a computer-generated result. The game provides instant feedback and tracks results over time.

## Features
- Player choice between head and tail.
- Random computer-generated coin result.
- Win and loss detection based on matching results.
- Score tracking for wins and losses.
- Persistent score storage using Local Storage.

## Technologies Used
- HTML
- CSS
- JavaScript
